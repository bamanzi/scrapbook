Tips for using GNU Emacs in MS Windows

(idea) by Frank Grimes Sun Feb 17 2002 at 15:58:44

First of all, download emacs from ftp://ftp.gnu.org/gnu/windows/emacs/

You MUST use the EMACS!

How To Set the GNU/EMACS's Default Font in MS Windows.
  1. find your .emacs file. Open it. Edit it. By default, it lives in your $HOME directory. You can set the $HOME variable using autoexec.bat (95/98/ME) or in Control Panel-->System-->Advanced--> Environment Variables. If you have no $HOME defined, it looks in c:\
  2. Shift-Left Click on emacs, and you'll get a windows font select dialog. Pick your favorite font.
  3. Go into the *scratch* buffer. Paste in the text:
    (insert (prin1-to-string (w32-select-font)))
    Now type C-x C-e. Select your font. Now text will mathamagically appear in the buffer. It will look something like:
    "-*-Courier New-*-*-*-*-17-102-120-120-c-*-iso8859-1"
  4. Now, go back to you .emacs file. Insert the line:
    (set-default-font "-*-Courier New-*-*-*-*-17-102-120-120-c-*-iso8859-1")

Use the console in Windows NT/2k/XP!

Normally when you type emacs -nw in a NT console, it doesn't work right. Try adding this line to your .emacs file:

(setq w32-use-full-screen-buffer nil)

(Of course you've added emacs to your $PATH. Right?)

Put Emacs in the sendto menu.

  1. Find your sendto folder. It might be in c:\windows\sendto\ or c:\documants and settings\***\SendTo\
  2. Put a shortcut there to the program
    <Wherever you installed emacs to>\bin\runemacs.exe
  3. Never use Notepad again!
